# CharityToken dApp

CharityToken is a decentralized application (dApp) that allows users to donate ERC20-like tokens to a designated charity address. It also enables users to transfer tokens between accounts. The application is built on the Ethereum blockchain and interacts with a Solidity smart contract.

---

## Features
1. **Token Balance Display**: Users can view their current balance of CharityTokens.
2. **Token Donation**: Users can donate tokens to a predefined charity wallet address.
3. **Token Transfer**: Users can transfer tokens to other Ethereum addresses.

---

## Prerequisites
1. [MetaMask](https://metamask.io/) installed and set up in your browser.
2. Node.js and npm installed for development (optional if hosting locally).
3. An Ethereum test network (e.g., Goerli, Sepolia) or a local blockchain (e.g., Ganache).
